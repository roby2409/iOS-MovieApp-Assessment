# iOS Movie Application

An iOS application built to explore and discover movies using The Movie Database (TMDB) API. This project is developed to demonstrate professional iOS development practices, focusing on clean architecture, reactive programming, and robust handling of network states.

## General Plans
- **Architecture:** MVVM (Model-View-ViewModel)
- **Features:** Core movie browsing, detailed views, and proper error handling.
